# Einführung

Diese Dokumentation richtet sich an alle Mitglieder der [Solidarischen Landwirtschaft Stadt, Land, Beides e. V.](https://stadt-land-beides.de/)

Grundlage für eine erfolgreiche [Solidarische Landwirtschaft](https://www.solidarische-landwirtschaft.org/startseite/) ist die Teilnahme und Zusammenarbeit von Mitmachern und Mitgliedern.
